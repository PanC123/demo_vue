技术栈

§ 技术栈

  • Vue 1.x
  • Vue Router 0.7.x
  • ES6 + Babel + ESLint 1.x
  • Lodash 4.x
  • Webpack 1.x（辅以 Gulp）
  • BrowserSync（多端调试）
  • Express 4.x（提供开发热更替与静态资源服务器）
  • jQuery + BootStrap 3.x (UI)

注意：

package.json 的 dependencies 中的：

  • lowdb
  • uuid
  • tiny-express

仅为留言板 Mock 服务器所需。同样地，devDependencies 中的：

  • gitbook-cli

仅为文档撰写所需
您往后的开发中，可以删掉：

  • 上述四个依赖
  • docs/ 与 mock/ 目录
  • src/services/xhr/index.js 中的 Mock Server 相关代码
  • package.json 中的 npm-scripts: build:docs
